# Batatabit Project

## Overview
Batatabit is a mobile-first cryptocurrency exchange landing page designed to demonstrate responsive web development principles. This project is built entirely with raw HTML and CSS, focusing on modern layout techniques and semantic structure without relying on external frameworks.

You can view the live project here: [Batatabit Live Demo](https://diegonaranjo94.github.io/Batatabit/)

## Features
- **Mobile-First Design**: The interface is designed primarily for mobile devices and adapts seamlessly to larger screens.
- **Modern CSS Layouts**:
  - Extensive use of **Flexbox** for flexible alignment (center, space-between, etc.).
  - Block model alignment techniques.
- **Styling**:
  - Custom font handling.
  - Variable usage for consistent theming.
  - Semantic HTML tags.

## Getting Started
To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/diegonaranjo94/Batatabit.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Batatabit
   ```
3. Open `index.html` in your preferred web browser.
